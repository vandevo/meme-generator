## Usage

1. Open `index.html` in a web browser
2. Choose an image:
   - Click "Choose image" to upload your own
   - Or select a template from the gallery
3. Add your text in the "Top text" and "Bottom text" fields
4. Customize:
   - Adjust text size with the slider
   - Change text and border colors
   - Drag text to position it anywhere
5. Click "Download" to save your meme

## Assets

The `assets/` folder contains stock photo templates that you can use as starting points for your memes.

## Technologies
